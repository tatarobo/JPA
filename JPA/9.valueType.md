

## Intro

 - JPA 데이터 타입
	 - 엔티티 타입
		 - @Entity 로 정의하는 객체
		 - 식별자를 통해 지속적인 추적 가능
	 - 값 타입
		 - int, Integer, String 같은 자바 기본 타입이나 객체
		 - 식별자가 없어서 추적 불가
		 - 분류
			 - 기본값 타입 (basic value type)
				 - 자바 기본 타입 (ex- int, double)
				 - 레퍼 클래스 (ex - Integer)
				 - String
			 - 임베디드 타입 (Embedded type 복합 값 타입)
				 - JPA 에서 사용자가 직접 정의한 값
			 - 컬렉션 값 타입 (Collection value type)
				 - 하나 이상의 값 타입을 저장할 때 사용
